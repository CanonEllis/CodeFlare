Learn: Array Operations

What are Array Operations?

NumPy allows you to perform element-wise operations on arrays, such as addition, subtraction, and multiplication. These operations are optimized for performance and are much faster than Python loops.

Examples to Try

  1. Element-wise Addition: 
    import numpy as np
array_1 = np.array([1, 2, 3])
array_2 = np.array([4, 5, 6])
result = array_1 + array_2
print("Addition:", result)
  2. Broadcasting: NumPy automatically broadcasts smaller arrays to match the shape of larger ones. 
    array = np.array([1, 2, 3])
scalar = 10
result = array + scalar
print("Broadcasting:", result)
  3. Common Mathematical Operations: 
    array = np.array([1, 2, 3])
print("Sum:", np.sum(array))
print("Mean:", np.mean(array))
print("Square Root:", np.sqrt(array))
