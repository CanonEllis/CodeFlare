Learn: Introduction to NumPy

What is NumPy?

NumPy is a Python library for numerical computations. It offers support for arrays, matrices, and mathematical operations.

Examples to Try

  1. Create a 1D Array: 
    import numpy as np
array_1d = np.array([1, 2, 3, 4, 5])
print("1D Array:", array_1d)
  2. Create a 2D Array: 
    array_2d = np.array([[1, 2, 3], [4, 5, 6]])
print("2D Array:\n", array_2d)
  3. Create an Array of Zeros: 
    zeros_array = np.zeros((2, 3))
print("Array of zeros:\n", zeros_array)
