Learn: Introduction to NumPy

What is NumPy?

NumPy is a powerful library in Python used for numerical computations. It provides support for arrays, matrices, and many mathematical functions.

Examples to Try

Follow these examples in the editor to understand how NumPy arrays work:

  1. Create a 1D Array: 
    import numpy as np
array_1d = np.array([1, 2, 3, 4, 5])
print("1D Array:", array_1d)
  2. Create a 2D Array: 
    array_2d = np.array([[1, 2, 3], [4, 5, 6]])
print("2D Array:\n", array_2d)
  3. Perform Element-Wise Addition: 
    added_array = array_1d + 10
print("Array after addition:", added_array)
  4. Reshape the Array: 
    reshaped_array = array_1d.reshape((5, 1))
print("Reshaped Array:\n", reshaped_array)
  5. Create an Array of Zeros: 
    zeros_array = np.zeros((2, 3))
print("Array of zeros:\n", zeros_array)
